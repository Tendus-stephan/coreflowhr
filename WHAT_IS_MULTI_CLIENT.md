# What is Multi-Client Management? (Simple Explanation)

## üéØ The Problem It Solves

### Scenario: You're a Recruitment Agency

**Without Multi-Client:**
```
Your Jobs Page:
- Software Engineer (TechCorp)
- Nurse (HealthCare Inc)
- Accountant (FinanceCo)
- Product Manager (TechCorp)
- Marketing Manager (HealthCare Inc)
```

**Problem:**
- All jobs are mixed together
- Can't filter by which company the job is for
- Can't see "all TechCorp jobs" at once
- Hard to organize when you have 10+ companies
- Can't generate reports per company

---

## ‚úÖ What Multi-Client Does

### It Lets You Organize Jobs by Company

**With Multi-Client:**
```
1. Create "TechCorp" as a client
2. Create "HealthCare Inc" as a client
3. When posting a job, select which client it's for
4. Filter jobs by client on the Jobs page
```

**Result:**
- Jobs are organized by company
- Filter: "Show me only TechCorp jobs"
- See all jobs for a specific company at once
- Better organization for agencies

---

## üí° How It Works

### Step 1: Create Clients
```
Go to Clients page ‚Üí Add Client
- Name: "TechCorp Inc"
- Email: contact@techcorp.com
- Phone: +1 (555) 123-4567
```

### Step 2: Link Jobs to Clients
```
When creating a job:
- Job Title: "Software Engineer"
- Client: Select "TechCorp Inc" from dropdown
- Save
```

### Step 3: Filter by Client
```
On Jobs page:
- Use "Client" filter dropdown
- Select "TechCorp Inc"
- See only TechCorp jobs
```
