# Force Browser to Load Latest Version

The logo image is deployed correctly, but your browser is showing a cached version. Here's how to force it to reload:

## Quick Fix (Hard Refresh):
1. **Windows/Linux:** Press `Ctrl + Shift + R` or `Ctrl + F5`
2. **Mac:** Press `Cmd + Shift + R`
3. This forces the browser to reload all assets

## Alternative Methods:

### Method 1: Clear Cache for This Site
1. Open Chrome DevTools (F12)
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"

### Method 2: Disable Cache (Temporary)
1. Open Chrome DevTools (F12)
2. Go to the Network tab
3. Check "Disable cache"
4. Keep DevTools open and refresh the page

### Method 3: Incognito/Private Window
1. Open an Incognito/Private window (Ctrl+Shift+N / Cmd+Shift+N)
2. Visit `www.coreflowhr.com`
3. This bypasses all cache

### Method 4: Clear Site Data
1. Open Chrome Settings
2. Go to Privacy and Security → Site Settings → View permissions and data stored across sites
3. Search for `coreflowhr.com`
4. Click it and select "Clear data"

## Verify the Image is Deployed:
The logo image is accessible at:
- `https://www.coreflowhr.com/assets/images/coreflow-logo.png`

You can test this URL directly in your browser to confirm the image loads.
